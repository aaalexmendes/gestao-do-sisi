Compartilhamento · JSONBin

Sincroniza os dados de 2026 (execução semanal + planejamento) com um bin no jsonbin.io, para que outras pessoas que abrirem este arquivo vejam os mesmos dados sem precisar fazer upload. Ao abrir, o dashboard tenta carregar do JSONBin em segundo plano; se estiver indisponível, usa os dados locais/embutidos. A chave fica salva apenas neste navegador e nunca é gravada no arquivo. Compatível com o bin do dashboard anterior (formato verba/categorias é reconhecido como planejamento). O planejamento embutido no HTML não é enviado ao bin (todos já o recebem com o arquivo publicado no GitHub Pages) — apenas execução, prazos e planejamentos de upload próprio são sincronizados. Planejamentos de upload são comprimidos automaticamente (gzip nativo do navegador, ~96 KB → ~20 KB), mantendo o envio abaixo dos 100 KB do plano gratuito.

Sem configuração do JSONBin neste navegador. O dashboard usa apenas dados locais.
